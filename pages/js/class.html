Class

I många fall vill man komma åt flera taggar som är relaterade, en viss Class. Och det är helt möjligt. Lika så kan man spara de taggarna som ett objekt, som beskrev tidigare. Detta görs genom document och en av dess funktioner.

var paragraphs = document.getElementsByClassName('thePs');

Nu finns det en variabel som heter paragraphs, i vilken alla taggar med den klassen finns. Genom den kan vi nu komma åt och manipulera taggarna. Vi kan ändra dess innehåll, eller dess utseende.

Vi kan t.ex. ändra texten i alla taggarna när man trycker på en knapp. Vi numrerar dem.

function changeText() {
    for (var i = 0; i < paragraphs.length; i++) {
        paragraphs.item(i).innerHTML = "Paragraf #" + i;
    }
}
<button onclick="changeText();">Byt text</button>

Då denna kod manipulerar saker i sidan, måste den läsas in efter att sidan har laddas. Därför placeras script-taggen sist i body-taggen

skript.js

var paragraphs = document.getElementsByClassName('thePs');

function changeText() {
    for (var i = 0; i < paragraphs.length; i++) {
        paragraphs.item(i).innerHTML = "Paragraf #" + i;
    }
}
