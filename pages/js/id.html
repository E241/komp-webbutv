ID

I många fall vill man komma åt en specifik tagg, ett visst ID. Och det är helt möjligt. LIka så kan man spara den taggen som ett objekt i en variabel. Detta görs genom document och en av dess funktioner.

var paragraph = document.getElementById('theParagraph');

Nu finns det en variabel som heter paragraph, i vilken taggen med id:t "theParagraph" finns. Genom den kan vi nu komma åt och manipulera taggen. Vi kan ändra dess innehåll, eller dess utseende.

Vi kan t.ex. ändra texten i en paragraf när man trycker på en knapp.

function changeText() {
    paragraph.innerHTML = "The quick brown fox jumps over the lazy dog";
}
<button onclick="changeText();">Byt text</button>

Då denna kod manipulerar saker i sidan, måste den läsas in efter att sidan har laddas. Därför placeras script-taggen sist i body-taggen

Färdig kod

index.html

<!DOCTYPE html>
<html>
    <head>
        <title>Sidans titel</title>
    </head>
    <body>
        <p id="theParagraph">Flygande bäckasiner söka hwila på mjuka tuvor</p>
        <button onclick="changeText();">Byt text</button>
        <script src="skript.js"></script>
    </body>
</html>

skript.js

var paragraph = document.getElementById('theParagraph');

function changeText() {
    paragraph.innerHTML = "The quick brown fox jumps over the lazy dog";
}

Resultat

Flygande bäckasiner söka hwila på mjuka tuvor
