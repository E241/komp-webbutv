Användarinteraktion

Här kommer ett kort exempel på hur man använder event-listeners, lysnnar efter användarinput och manipulerar en tagg efter användarens input. Försök förstå hur koden fungerar och prova dig fram själv. Ändra koden och se vad som händer. Du kan alltid ta hjälp utav W3Schools om du kör fast.

Färdig kod

index.html

<!DOCTYPE html>
<html>
    <head>
        <title>Sidans titel</title>
    </head>
    <body>
        <h2>Du har skrivit: <a id="output"></a></h2>
        <input type="search" oninput="updateText();" id="input"></input>
        <script src="skript.js"></script>
    </body>
</html>

skript.js

var output = document.getElementById('output');
var input = document.getElementById('input');

function updateText(e) {
    output.innerHTML = input.value;
}
