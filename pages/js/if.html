If-statement

Som det är nu, kan vi inte anpassa programmet beroende på vad variabler har för värde. Men det kan vi ändra på. En av de viktigaste sakerna i alla programspråk är if-statements. Det är en grundläggande del i boolesk algebra som kollar om ett uttryck är sant eller falskt.

Den enklaste varianten är att kolla om en variabel är lika med ett tal.

if (fish == 10) {
    alert('Fish = 10');
}

Som ni såg skrev jag == istället för =. Det är för att ett likamedtecken betyder att jag ger en variabel ett värde, och två likamedtecken betyder att jag kollar om de är lika med varandra.

Man kan även kolla om en variabel inte är lika med ett tal.

if (fish != 10) {
    alert('Fish är inte 10');
}

De flesta andra matematiska kontroller fungerar också, och finns listade nedan.

if (fish > 10) {
    alert('Fish är större än 10');
}

if (fish >= 10) {
    alert('Fish är större än eller lika med 10');
}

if (fish < 10) {
    alert('Fish är mindre än 10');
}

if (fish <= 10) {
    alert('Fish är mindre än eller lika med 10');
}

Utöver de matematiska kontrollerna finns det några andra som är viktiga att kunna.

//Kollar om en boolean är true, eller om det är en annan typ huruvida den har ett värde.
if (fish) {
    alert('Sant');
}

//Motsatsen till den förra
if (!fish) {
    alert('Falskt');
}

//Kollar om fish har samma värde som 10 och är av samma typ
if (fish === 10) {
    alert('Fish = 10');
}
//Kollar om fish inte har samma värde som 10 eller inte är av samma typ
if (fish !== 10) {
    alert('Fish är mindre än eller lika med 10');
}

Ibland vill man kolla flera värden innan man gör något, tillexempel att ett tal är större än noll men mindre än eller lika med 10. Det behöver man inte göra flera if-statements för att kolla, utan kan kollas i ett. Man kan även kolla om antingen det ena eller det andra är sant. T.ex. om ett tal är lika med 10 eller är lika med 20.

//Kollar om båda uttrycken är sanna
if (x > 0 && x <= 10) {
    alert('Sant');
}

//Kollar om minst ett av utrycken är sant
if (x == 10 || x== 20) {
    alert('x = 10 eller 20');
}
