Funktioner

Som du har sätt tidigare så har vi skrivit alert('Text'); i skripten. Då kallar vi p en funktion. Funktioner är egentligen kodstycken, som en maskin som tar några invärden, eller inga, kör dem genom en utför något, och skickar ibland tillbaka något. De är en viktig del i JavaScript.

En funktion består av en serie instruktioner i form av funktionsanrop och matematiska operationer på variabler. Även sammanslagning och annan manipulering utav variabler ingår i funktioner.

En funktion måste definieras med nyckelordet function samt ges ett namn. Den kan även ges ett antal namn på argument, indata.

function funktionsNamn(arg1, arg2, ...) {
    //Funktionens innehåll
}

Vi gör en exempelfunktion som ska köras när användaren trycker på en knapp. När knappen blir tryckt på ska ett meddelande visas för användaren som berättar hur många gånger användaren tryckt på knappen. Först behöver vi definiera de variabler som vi använder. Sedan skriver vi funktionen.

var count = 0;
function knappTryck() {
    count++; //Samma som count = count + 1
    alert('Du har tryckt ' + count + ' gånger!');
}

Sedan skapar vi en knapp i HTML som kallar på våran funktion.

<button onclick="knappTryck();">Tryck här!</button>

<!DOCTYPE html>
