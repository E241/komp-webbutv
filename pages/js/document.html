Document

Fram tills nu har JavaScriptet inte varit så intergrerat i webbsidan, vi har gått igenom mestadels grunder. Dessa grunder är dock nödvändiga för att kunna manipulera webbsidor med hjälp av JavaScript. Det som gör att JavaScript kan manupilera en sida är DOM, Document Object Model. Den HTML-kod du skrivit omvandlas utav webbläsaren till ett DOM-träd. Detta DOM-träd kan sedan ändras av webbläsaren på valfritt sätt, delvis genom JavaScript.

För att komma åt detta DOM-träd används objektet document. Där i finns all information och alla objekt som finns i webbsidan. Vi kan t.ex. ändra sidans titel.

document.title = "Ny titel!";

Utöver document finns objektet window och location som båda ger ytterligare kontroll över webbläsaren, så som dess storlek och vad den är inne på för adress.

I kommade guider kommer vi att berätta grundläggande delar av hur man manipulerar sidan med hjälp utav JavaScript.
