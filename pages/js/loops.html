While & For

Ibland vill man göra någonting så länge ett uttryck stämmer, t.ex fortsätta fråga efter gissningar tills användaren gissat rätt. Och det kan man göra med en while-loop. Den ser ut ungefär som ett if-statement, den bara kör koden om och om igen så länge uttrycket stämmer.

while (x != 2) {
    //Det som ska köras så länge uttrycket stämmer
}

Alla kontroller som går att göra med ett if-statement gårt även att göra med en while-loop.

Det finns även en variant på en while-loop, i fall man vill att koden ska köras minst en gång, trots att uttrycket inte stämmer.

do {
    //Det som ska köras så länge uttrycket stämmer och minst en gång även om det inte stämmer
} while (x != 2);

Sedan har vi for-loop. Den iterear över, gå igenom alla värden, en array. Den kan även användas för annat, men det är det vanligaste användningsområdet. En for-loop tar dock 3 statements. En som körs innan loopen, ett som kollar om loopen ska köras och slutligen en som körs efter loopen. Vanligtvis brukar det första statementet initiera en variabel, det andra kolla hur variabeln relaterar till någon annan, och slutligen ökas eller minskas värdet på variabeln.

var cars = ['BMW', 'Toyota', 'Skoda', 'Porsche'];
for (var i = 0; i < cars.length; i++) {
    alert(cars[i]); //För att få tag på variabeln används cars[i] där i är iterationen
}

Det finns öven en variant av for-loop som itererar över ett object. Den kallas för for/in-loop.

var object = {key1:'värde 1', key2:'Nyckel 2', keyNr:3};
for (x in object) {
    //Det som ska göras med x
}
