Länkar

Webben bygger på en liten, men väldigt viktig sak, nämligen länkar. Det är länkar som gör så att man kan hitta nya sidor, ta sig dit man vill eller dela med sig utav sina upptäckter. Utan länkar skulle sidor som google vara rätt meningslösa. Att navigera på hemsidor skulle vara näst intill omöjligt.

Länkar definieras med hjälp utav a-taggen. Och precis som img-taggen accepterar den olika attribut. Antalet attribut på a-taggen är enormt, men vi kommer bara gå igenom det som används till länkar. Nämligen href-attributet.

<a href="http://www.google.se">Länktexten</a>

Innanför a-taggen kan man skriva fler taggar, som t.ex. h1-taggen. Det gör att du kan göra även bilder till länkar.

<a href="http://www.google.se"><h1>Stor text</h1></a>
<a href="http://www.google.se"><img src="bild.jpg" alt="Länkbild" /></a>

Färdig kod

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>Sidans titel</title>
    </head>

        <a href="http://www.google.se">Länktexten</a>
        <a href="http://www.google.se"><h1>Stor text</h1></a>

    </body>
