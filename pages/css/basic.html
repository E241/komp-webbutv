Så nu kan vi sätta igång med skälva css koden.

Css är uppbyggt på ett mycket enkelt sätt. Först väljer du ett objekt du vill ____, till exempel paragrafer (<p>) 

 p

Efter objektet du vill ändra så skriver du "curlybrackes" eller "klammer paraneser" ({}). Man brukar lägga ut dem såhär. 

 p {
                
            }

Sen är det bara att skriva in dina css komandon innanför paranteserna. Två exempel som jag tänkte visa var "color" och "background-color".

 p {
    color:white;
    background-color:red;
        }

Color är färgen på texten och background-color är färgen på bakgrunden. Kommer komma in på färger nästa.
