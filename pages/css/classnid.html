Classes & Id:s

Ibland vill man inte ändra alla paragrafer eller alla divs utan bara några, då är klasser och id:s bra att kunna.

Class

Vi kan börja med klasser.
Klasser eller classes som det heter på engelska är gruperingar av objekt, det är som en markering som du kan lägga på flera objekt för att ändra dem som grupp.

Du markerar ett objekt som en viss klass igenom att skriva class="classname".
(classname byts ut mot ditt klassnamn som du väljer skälv).

<div class="clasname">
    <p class="classname"></p>
</div>

Du kallar sedan på klassen i css dokumentet igenom att skriva ".classname".

 .classname {
    color: red;
}

Du kan även ha flera klasser på ett objekt igenom att bara sätta ett mellanslag.


    <div class="classname1 classname2"><div>

Id

Id:s funkar på ungefär samma sätt bara att ett id bara definerar ett objekt, så det kan till exempel användas för att separera objekt av samma klass.

Du anger ett id på samma sätt som med class bara att du skriver id="idname".


    <p class="classname" id="idname1">Hej alla barn</p>
    <p class="classname" id="idname2">Nu är det fredag!</p>

Du kallar på ett id på samma sätt som klass fast med "#" itsället för ".".

.classname{
    color:red;
}
#idname1{
    background-color:#DDEE22;
}
#idname2{
    background-color:rgb(0,225,225);
}

Result

Hej alla barn

Nu är det fredag!
