Andra selektorer

Det finns flera sätt att välja ett specifikt objekt i html/css. Kom ihåg att dethär är bara några exempel.

Specifik id/class

Du kan specifisera villket slags objekt med en class eller ett id du vill välja.

 div#idname {
    
    color: red;
    
}
            
p.classname{


    background-color:#FFFFFF;

}

Parent and child

Parent betyder förelder och child betyder barn, men vad har det med html och css att göra? Jo det ska jag berätta.

Om en <p> tag är direkt innanför en <div> som visas under tag är <div> taggen parent till <p> och <p> tagen är child.


    <div>
        <p> Den här paragrafen är direkt "child" till div taggen </p>
        <span>
            <p> Den här paragrafen är inte direkt "child" till div taggen </p>
        </span> 
    </div>

Om du vill ändra alla paragrafer som är direkt child till en div kan du göra div>p.


div>p {
    background-color:blue;
}

Om du vill ändra alla paragrafer som befinner sig i en div men inte behöver vara direkt "child" kan du använda div p.


div p {
    color:green;
}

Flera selektors

Om du vill ha samma css kod för en klass, ett id och alla paragrafer går det att göra bara igenom att sätta ett komma.


.classname, #idname, p{
    color: green;
}
