Färger

Det finns tre olika "main" sätt att göra färger i html/css.

Det första och det vanligaste sättet är med heximala tal.
Hexadecimala tal har basen 16 och har, till skillnad från decimala tal (bas 10), 15 siffror (1,2,3,4,5,6,7,8,9,a,b,c,d,e,f).
Sex hexatal bakom en '#' bildar en färg. 

 body{
            
                color: #FFFFFF;
                color: #4B0082;    
                color: #800517;
                color: #000000;
            }

Det tredje är och på sätt och vis det enklaste är att man advänder de prefixer som redan finns i html. 

 body{
                /* Dessa exempel motsvarar den innan */
                color: white;
                color: indigo;    
                color: firebrick;
                color: black;
            }

För en lista på alla namn prefix Clicka här.

Det tredje sättet kallas för rgb eller rgba. rgb står för (Röd, Green, Blue) villket är de tre huvud färgerna i våran färg vidd. Du blandar de tre grund färgerna och max är 255 och min är 0. Skillndaden mellan rgb och rgba är att rgba har en opacity faktor som går från 0 till 1 så du kan ha halvt genomskinliga färger. 

 body{
                /* Dessa exempel motsvarar de innan */
                color: rgb(255,255,255);
                color: rgb(75,0,130);    
                color: rgb(128,5,23);
                color: rgb(0,0,0);
                
                color: rgba(255,0,0,0.5); /* En halvt genomskinlig röd */
            }

För att välja färger och se deras hex och rgb värden kan du gä in på Colorpicker.com
