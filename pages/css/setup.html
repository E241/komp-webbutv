Setup

Det finns två sätt att implementera css kod i html.

Det ena sättet är att man skriver css koden emellan "style" taggar. Style taggarna ska vara imällan "head" taggarna i html dokumentet. 

<!DOCTYPE html>
<html>
    <head>
        <style>
            Insert css here
        </style>
    </head>
</html>

Detta går bra för mindre hemsidor med en till två html filer men ju större hemsidan blir destå opraktiskare blir det.

Det andra alternativet är att man andvänder en Css fil (.css) och kallar då på den med en link tagg. 


<html>
    <head>
        <link rel="stylesheet" href="filnamn.css">
    </head>
</html>

Fördelarna med att använda en .css fil är att det blir mer organiserat och du kan andvända samma css fil till flera sidor.
